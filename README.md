# Wetter App

Eine einfache Webanwendung, mit der Benutzer das aktuelle Wetter für eine Stadt abrufen können.  
Die App verwendet eine öffentliche Wetter-API, um aktuelle Wetterdaten wie Temperatur und Wetterbedingungen anzuzeigen.

Dieses Projekt wurde erstellt, um den Umgang mit APIs, JavaScript und der dynamischen Darstellung von Daten zu demonstrieren.

## Technologien

- HTML
- CSS
- JavaScript
- Wetter API (z.B. OpenWeatherMap)

## Funktionen

- Stadtname eingeben
- Aktuelles Wetter anzeigen
- Temperatur anzeigen
- Wetterbeschreibung anzeigen
- Einfache und benutzerfreundliche Oberfläche

## Nutzung

1. Projekt herunterladen oder klonen.
2. Alle Dateien im selben Ordner speichern.
3. `index.html` im Browser öffnen.
4. Eine Stadt eingeben und das aktuelle Wetter abrufen.
